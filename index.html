Curso Programador WEB

Sobre o curso de programador WEB

Vocé aprenderé a planejar, programar e publicar websites com métodos que permitem a inserção e manipulação de conteúdo
dinâmico, além de desenvolver interfaces de acordo com os padrões web.

HTML

O HTML éa linguagem base para se desenvolver qualquer site. É uma linguagem baseada em marcação, onde marcamos os
elementos para definir quais informações a página vai exibir. A linguagem HTML foi criada na década de 1990 e suas especificagdes sio controladas pela W3C (World Wide Web Consortium).
